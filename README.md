# SQL Refresher – Advanced Movie Database

A fully advanced SQL project to demonstrate mastery of relational database concepts. Includes movies, actors, reviews, box office data, and roles.

## Skills Demonstrated
- Table creation with primary & foreign keys
- Joins, aggregates, filtering, and group queries
- Views, triggers, stored procedures
- Transactions and data integrity
- Sample data for testing queries

## Folder Structure
- `01_create_tables.sql` – Define all tables and relationships
- `02_insert_data.sql` – Insert sample data
- `03_queries.sql` – Practice queries demonstrating key SQL concepts
- `04_triggers_views_procedures.sql` – Advanced features for automation and analytics

## Optional Visuals
Include an ER diagram (`ER_diagram.png`) showing relationships between tables.

## How to Run
1. Execute `01_create_tables.sql`  
2. Execute `02_insert_data.sql`  
3. Test queries in `03_queries.sql`  
4. Run advanced features in `04_triggers_views_procedures.sql`
